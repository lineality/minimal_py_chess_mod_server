  Hygiene

This site is designed to be as simple and private

as possible, using only HTML, no java-script, no CORS,

no cross-site anything, no user-analytics, no wordpress, no plugins, no bloat.

No obligatory registration with email, name, password, etc.

In order for your game to be private (and not be watched and interfered with by

literally anyone on the internet,

game admission needs to be restricted to ip connections

specified during game-setup. To be part such of a private game

you need to identify yourself

somehow. This can be done without any of your information stored or collected.

No information about players is collected.

A cookie which contains only the fact that you

are allowed to play this game is stored in your browser for a duration set during

game-setup (the maximum time-length that the game can go on for).

The cookie contains a unique timestamp player nickname to record who made what moves

A hash

of that cookie is kept in the server game-file to verify

that you were one of the people allowed to join the game.

If you want to set up a clone of this server with more or less privacy

(though more may be hard to achieve),

by all means start a fun maker-project and launch your own server.
